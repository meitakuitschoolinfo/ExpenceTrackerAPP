# 📐 第8章 メッセージ一覧

[← 目次に戻る](./README.md)

画面に表示するメッセージを一覧化する。区分は次の3つ。

- **フラッシュ**：更新成功/失敗をリダイレクト先で1回だけ表示（緑＝`message` / 赤＝`errorMessage`）
- **項目エラー**：バリデーション違反を該当項目の下に表示
- **全体エラー**：フォーム全体のエラー（重複等）をフォーム上部に表示（`globalErrors`）

---

## 8-1. フラッシュメッセージ（成功通知）

| ID     | 文言                                       | 表示色 | 発生機能 |
| ------ | ------------------------------------------ | ------ | -------- |
| MSG-S01 | 登録が完了しました。ログインしてください。 | 緑     | FNC-05 |
| MSG-S02 | 記録を登録しました                         | 緑     | FNC-09 |
| MSG-S03 | 記録を削除しました                         | 緑     | FNC-10 |
| MSG-S04 | カテゴリーを登録しました                   | 緑     | FNC-13 |
| MSG-S05 | カテゴリーを更新しました                   | 緑     | FNC-15 |
| MSG-S06 | カテゴリーを削除しました                   | 緑     | FNC-16 |

---

## 8-2. フラッシュメッセージ（エラー通知）

| ID     | 文言（例）                                                  | 表示色 | 発生機能 |
| ------ | ----------------------------------------------------------- | ------ | -------- |
| MSG-E01 | このカテゴリーは N 件の記録で使われているため削除できません | 赤     | FNC-16（使用中削除） |

---

## 8-3. 認証関連メッセージ（ログイン画面）

| ID     | 文言                                       | トリガ           |
| ------ | ------------------------------------------ | ---------------- |
| MSG-A01 | メールアドレスまたはパスワードが間違っています | `/login?error`   |
| MSG-A02 | ログアウトしました                         | `/login?logout`  |

---

## 8-4. 業務エラーメッセージ（globalErrors）

Service が投げ、Controller が `bindingResult.reject` でフォーム全体エラーとして表示。

| ID     | 文言（例）                                       | 発生機能 | 例外 |
| ------ | ------------------------------------------------ | -------- | ---- |
| MSG-G01 | このメールアドレスは既に登録されています: xxx    | FNC-05   | IllegalArgumentException |
| MSG-G02 | 同じ名前のカテゴリーが既に存在します: xxx        | FNC-13/15 | IllegalArgumentException |
| MSG-G03 | カテゴリーの種類が記録の種類と一致しません        | FNC-09   | IllegalArgumentException |

---

## 8-5. バリデーションメッセージ（項目エラー）

| ID     | 文言                                       | 対象項目 |
| ------ | ------------------------------------------ | -------- |
| MSG-V01 | メールアドレスは必須です                   | email（Login/Register） |
| MSG-V02 | メールアドレスの形式が不正です             | email |
| MSG-V03 | メールアドレスは255文字以内で入力してください | email（Register） |
| MSG-V04 | パスワードは必須です                       | password |
| MSG-V05 | パスワードは6文字以上で入力してください     | password（Register） |
| MSG-V06 | ユーザー名は必須です                       | name |
| MSG-V07 | ユーザー名は100文字以内で入力してください   | name |
| MSG-V08 | 種類（支出/収入）は必須です                | type（Category/Transaction） |
| MSG-V09 | カテゴリー名は必須です                     | label |
| MSG-V10 | カテゴリー名は100文字以内で入力してください | label |
| MSG-V11 | 色は必須です                               | color |
| MSG-V12 | 色の指定が不正です                         | color |
| MSG-V13 | 日付は必須です                             | transactionDate |
| MSG-V14 | カテゴリーは必須です                       | categoryId |
| MSG-V15 | 金額は必須です                             | amount |
| MSG-V16 | 金額は1円以上で入力してください            | amount |
| MSG-V17 | メモは255文字以内で入力してください        | memo |

---

## 8-6. 確認ダイアログ（JavaScript confirm）

| ID     | 文言                       | 発生箇所 |
| ------ | -------------------------- | -------- |
| MSG-C01 | この記録を削除しますか？   | 履歴一覧の削除（FNC-10） |
| MSG-C02 | このカテゴリーを削除しますか？ | カテゴリー一覧の削除（FNC-16） |

---

## 8-7. 空データ時の案内

| 画面            | 文言 |
| --------------- | ---- |
| ダッシュボード（内訳0件） | 支出データがありません |
| 履歴一覧（0件） | 記録がありません／この月のデータはまだ登録されていません。 |
| 記録フォーム（カテゴリー0件） | カテゴリーがありません。設定画面から追加してください。 |
